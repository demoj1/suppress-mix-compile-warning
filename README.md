Script for suppressing many useless warning, generated by mix.

```bash
git clone git@github.com:demoj1/suppress-mix-compile-warning.git && \
cd suppress-mix-compile-warning && \
chmod u+x mixx
```

Run mixx instead mix, example:

```
mixx compile
```
